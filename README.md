# Conecta Dev - Jackson Luiz

**Conecta Dev** é uma central de links (Link-in-bio) inovadora, desenvolvida para unificar e apresentar minhas redes sociais e projetos em um único lugar.

Inspirado no lendário projeto do **Ignite da Rocketseat**, esta versão foi elevada ao próximo nível através do design **"Nebula Glass"**.

![Preview do Projeto](https://img.shields.io/badge/Status-Vers%C3%A3o%20Criativa%20(Nebula%20Glass)-blueviolet?style=for-the-badge&logoColor=white)

## Destaques do Design "Nebula Glass"

Este projeto utiliza o que há de mais moderno em UI/UX para uma experiência imersiva e responsiva:

-   ** Background Radiante**: A imagem de fundo galáctica (`bg-mobile.jpg`) foi mantida como a estrela e fonte de luz e atmosfera para todo o design.
-   ** Autêntico Glassmorphism**: Os botões sociais agora são cartões largos de vidro (`backdrop-blur-md`, texturas brancas super translúcidas) que distorcem o fundo, criando um visual flutuante e orgânico.
-   ** Brand Hover FX**: Ao passar o mouse em um botão (Ex: WhatsApp, LinkedIn), o cartão não apenas infla na tela, mas ganha um brilho sutil (box-shadow glow) irradiando a **exata cor da marca** selecionada.
-   ** Micro-Interações Vivas**: O avatar de foto possui um `ring glow` gradiente dinâmico rotacionando em segundo plano e dá um "pulo" suave quando focado.
-   ** Dark/Light Mode Nativo**: Transição suave (0.8s globais) ativada por um elegante botão switcher com física elástica/bezier (bounce).

## Tecnologias Utilizadas

-   **HTML5 Semântico**: Estrutura sólida, minimalista e totalmente acessível (ARIA attributes inclusos para leitores de tela avançados).
-   **Tailwind CSS (CDN)**: Construção modular visual e otimizada que faz uso extensivo do novo JIT de valores arbitrários complexos diretos no HTML.
-   **JavaScript Vanilla**: Interatividade de estado global para controlar o Dark/Light theme lendo as raízes do HTML, mantendo-se fiel ao script original do projeto.
-   **Ionicons**: Biblioteca vetorial SVG open-source hiper leve e escalável.
-   **Google Fonts**: Tipografia `Inter`, a queridinha universal da melhor legibilidade.

## Como Executar

1. Clone o repositório em sua máquina:
    ```bash
    git clone https://github.com/JacksonLuiz99/conecta-dev-jackson.git
    ```
2. Abra a pasta e clique duas vezes no arquivo `index.html` em qualquer navegador (de preferência atualizado, que suporte CSS Backdrop Filters). Não é necessário rodar Node ou Npm.
